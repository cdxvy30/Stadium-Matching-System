# API Docs

## User related API

### Sign Up API

API URL: /api/user/signup

Method: POST

response type: application/json

Input Format
```
{
    email: string,
    password: string,
    name: string
    age: int
    gender : string
    badminton : int,
    basketball: int,
    volleyball : int,
    self_intro : str

}
```

Response Format
```
{
    "user_id" : int
}
```

### Sign in API

API URL: /api/user/signin

Method: POST

response response type: application/json

Input Format
```
{
    email: string,
    password: string,
}
```

Response Format
```
{
    user_id : int
}
```

### Personal Profile Update API

API URL: /api/user/profile

Method: PUT

response response type: application/json

Header
```
{
    Authorization: "Bearer ${JWT}"
}
```

Input Format
```
{
    name: string,
    email : string
    self_intro: string
    badminton : int,
    basketball: int,
    volleyball : int,

    baseball: int,
    tennis: int,
    swimming: int,
    gym: int
}
```

Response Format
```
{
    user_id: int
}
```
